Tools used in this project

Requirements

  • Ncurses

Install Ncurses (apt)

apt install libncurses5-dev libncursesw5-dev

Alternatively, install all tools used in this project with:

make install

<details>

  • Ncurses: Window management, required to play
  • GDB: Debugger
  • Valgrind: Memory management tool
  • Doxygen: Documentation generator

</details>

Play

make
./jogo
